## Summary
Team Finance is a multi-chain token vesting and liquidity locking platform targeting project teams and early-stage token launches. It combines vesting contracts, LP token locks, and tokenomics management in one product suite. Notable for "branded vesting" — embeddable vesting widgets for project websites.

## Key Facts
- **Chains supported:** BSC, Ethereum, Avalanche, Polygon, and more
- **Core products:**
  - Token vesting contracts (customizable schedules)
  - LP token locks (liquidity lock for rug-pull prevention)
  - Token launch (create + distribute tokens)
  - Tokenomics management (supply, taxation, burn, mint)
- **Branded vesting:** Embeddable widget — projects embed their vesting dashboard directly on their website
- **Audited contracts:** Same smart contracts used in main dashboard and embedded widgets
- **Target users:** Early-stage projects, launchpads, DeFi protocols needing credibility signals
- **Trust signals:** Token locks + vesting together = strong anti-rug-pull posture

## Branded Vesting Feature
The vesting widget allows projects to:
- Embed vesting info on project website (own the claiming UX)
- Display tokenomics alongside roadmap milestones
- Serve investor/community documentation within their own brand
- Widget uses the same audited smart contracts as the main platform

> Team Finance appears more focused on early-stage token projects (pre-IDO credibility) vs. Sablier which targets established DAOs and protocols. The LP lock + vesting combo is a common launchpad requirement.

## How it relates to Logos
